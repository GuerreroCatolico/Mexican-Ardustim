MEXICAN ARTHUR-STIM esta diseñado para poder hacer un generador ckp, cmp1, cmp2, en base de arduino uno con lcdkeypadshield, sin necesidad de un set up costoso. este proyecto esta basado en los deseños ya existentes de ardustim y el proyecto de copperhead donde genero un ardustim con lcdkeypadshieldx asi que se juntan ambos proyectos y se genera uno con los dos beneficios y ademas se agregan señales para el mercado mexicano ya que el proyecto esta diseñado en general para el mercado Mexicano.

Materiales:
arduino uno, cualquier version esta bien incluso la clone
